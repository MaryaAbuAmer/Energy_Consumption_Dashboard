# Energy Consumption Analysis Dashboard

## Project Overview
This project analyzes energy consumption and climate-related indicators across multiple countries from 2020 to 2024.

The analysis focuses on:
- Energy consumption patterns
- Peak demand periods
- Relationship between temperature and energy usage
- Renewable energy share
